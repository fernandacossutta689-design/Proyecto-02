# Proyecto-02

Consigna: 
Crear ua pagina web utilizando bootstrap y html. Elegir una tematica a elecciòn y utilizar 10 componentes de la libreria bootstrap.

Lo primero que yo hice fue pensar de que va a ser mi pagina y que componentes iba a utilizar

1-Use un navbar para poner una barra de navegacion principal, para que el usuario se pueda ubicar

2-Use un carousel para poner imagenes de los platos o promociones que haya 

3-Use cards para mostra la imagen de un plato y agregarle informacion como los vegetale que lleva, etc

4-Use un list group para mostrar las categorias o tipos de platos que hay

5-Use buttons para poner Reservas o el "Ver más detalles"

6-Use forms para poner los formularios donde ingresen sus datos el usuario 

7-Use alerts para que aparescan avisos de cosas importantes como "Reserva confirmada"

8-Use modal para confirmar los pasos que hiciste como "confirmar reserva"

9-Use un grid sistem para que la pagina se adapte a cualquier medio (tablet, celular, computadora, etc)

10-Use un footer para poner el contacto, redes y el nombre de la pagina.
